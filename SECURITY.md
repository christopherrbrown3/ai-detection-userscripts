# Security and privacy

## Privacy boundary

The installed userscripts analyze visible post text in the current browser tab. They make no network requests, load no remote code, and store only per-site display settings in local storage.

The offline training tools read only paths explicitly supplied on the command line and write only to the requested output paths.

## Reporting

Please report a suspected privacy leak, unintended network request, code-injection issue, or unsafe content extraction privately to the repository owner through GitHub's security-reporting feature when available. Do not attach private posts, messages, cookies, tokens, or browsing history.

Ordinary site-selector breakage and badge-placement problems can be reported through public issues with sanitized fixtures.

## High-stakes use

The output is not suitable for academic discipline, employment decisions, moderation enforcement, fraud findings, or other high-stakes authorship judgments. It is an experimental local style signal only.
